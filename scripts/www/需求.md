## 需求
1、本系统是一个儿童跳绳计时、计数系统。
1、 js + css + html 写在同一个文件。
2、本次输出指产出一个HTML文件。
3、功能要严谨。

## 功能

1、拥有一个计时器（可选择 1min 、5min、10min、15min、20min、25min、30min、自定义）。
2、拥有儿童、姓名头像展示与修改功能。
3、可生成本次计时、计数签名卡
4、签名卡：儿童信息 + 时间（YYYY-MM-DD hh:mm:ss） + 跳绳分钟数 + 中断次数

## 优化
1、暂停时 自动加一次中断次数
2、增加 累计倒计时次数 可手动控制（共计跳绳多少分钟、刻度为1）
